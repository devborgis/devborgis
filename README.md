## 👨‍💻 About Me

I'm a passionate programmer, educator, and supervisor with a keen interest in Object Pascal (Delphi and Lazarus) and database management. My focus is on back-end processes and the logic that powers applications, making them efficient and robust. As the co-founder of **Spote Brasil** and a **Supervisor** and **Programador at CRTI Soluções**, I work on projects that allow me to improve processes and create reliable tools for enterprise use.

- 🔸 **Current Roles**:

  - Supervisor and Programador at **CRTI Soluções**
- 🎯 **Main Focus**: Object Pascal, Node.js, and database management
- 💡 **Interests**: Efficient algorithms, custom tool development, and database optimization
